I'm Biffer Rowley — a full-stack engineer based in East Sussex with a deep focus on artificial intelligence, real-time data systems, and immersive 3D visualisation.

I build products that sit at the intersection of AI and human experience — from social intelligence engines that map fandoms in 3D space, to flight monitoring systems tracking every aircraft in the sky, to autonomous trading systems powered by machine learning.

My philosophy is simple: complex technology should feel effortless. Every system I build prioritises clarity, performance, and an interface that makes sophisticated data accessible to everyone.

Technical Deep Dive

A closer look at the engineering decisions and architecture behind each system.

🗺️

Fandom Mapper

A social intelligence platform that turns raw social media data into interactive 3D knowledge graphs, revealing hidden patterns in brand-creator relationships.

Frontend
  • React 19 with Vite for instant HMR
  • react-force-graph-3d for physics-based 3D node graphs
  • TailwindCSS utility-first component styling
  • Custom Visual DNA widget rendering
Backend & Data
  • Node.js + Express REST API
  • MongoDB with chunked split-storage for >16MB datasets
  • Apify custom actors for Instagram/TikTok scraping
  • Anti-hallucination data integrity system
AI / ML
  • Google Gemini 3 Pro / 3.0 Flash
  • Enhanced provenance reasoning engine
  • Autonomous multi-hop Wizard orchestration
  • Sentiment & content analysis pipelines
Infrastructure
  • Google Cloud Run containerized deployment
  • Docker multi-stage builds
  • MongoDB Atlas for managed scalability

Nyx

A real-time flight intelligence engine combining live ADSB telemetry, scraped incident data, and a graph database into a unified safety analysis platform.

Frontend
  • React + Vite with CSS-in-JS
  • Three.js custom 3D globe renderer
  • Physics-based movement interpolation
  • HUD overlays with atmospheric gradient fades
Backend & Data
  • Node.js + TypeScript pipeline
  • Neo4j graph database for relational mapping
  • SQLite local cache + audit logging
  • ADSB.lol live telemetry integration
Intelligence
  • Source Authority Ranking (SAR) system
  • Reconciliation engine for conflicting data
  • Aviation Herald + ICAO/FAA/NTSB scraping
  • Pattern analysis for fleet & regional risks
Performance
  • Sticky quota management for 5,000+ aircraft
  • Critical priority bypass for emergency squawks
  • Mesh recycling for render stability
