# Biffer Rowley — Portfolio

Personal portfolio website showcasing AI/ML engineering, 3D visualization, and real-time intelligence systems.

## Projects Featured

- **Fandom Mapper** — AI-powered social intelligence engine with 3D force graphs
- **Nyx** — Flight Intelligence Engine for real-time aviation monitoring
- **Crypto Trader** — AI Strategy Generator & Autotrader

## Tech

Single-page site built with vanilla HTML, CSS, and JavaScript. No build step required.

- Space Grotesk + JetBrains Mono typography
- Dark theme with ambient glow effects
- Intersection Observer scroll animations
- Procedurally generated screen mockups (3D graph, globe, candlestick chart)
- Fully responsive

## Deploy

### GitHub Pages

1. Go to **Settings → Pages**
2. Set source to **Deploy from a branch**
3. Select `main` branch and `/ (root)` folder
4. Save — site will be live at `https://vanillabrand.github.io/portfolio/`

### Any Static Host

Just serve `index.html`. No dependencies, no build step.
